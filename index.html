PROJECTS

  • TabCode: Given a Javascript library without a formal API, I analyzed the code and repackaged it with some examples that I created. The goal was to make it possible for inexperienced users to display lute tablature on the web.

  • LaClaveciniste.Com: I was given the design for a small site and rebuilt it using Rails and Bootstrap. The main task here was to take a design created on a Macbook and rebuild the HTML in a sensible way with Bootstrap. In terms of Rails, the highlight is the internationalization (I18n) with support for two languages (Estonian and English).

  • Radiogetaway.Com: I created the design and built this site using Rails and Bootstrap. The site uses the Devise and Omniauth gems and it is deployed on Heroku.

  • Estonian Spidering Project: As part of my studies of the Estonian language, the goal of this project was to develop a Ruby class that would spider the linguistic website http://www.filosoft.ee/gene_et/ and save the data on the local server. Environment: Ruby on Rails. Main focus: MECHANIZE and PG_SEARCH gems.

  • Garage48 project: During this 48-hour startup weekend, I was the backend developer for the 1st place team. Working with a human trafficking expert from an NGO, we created an app that regularly spiders a website offering sexual services and allows his team to analyze the nature of the offered services. I have continued working on this project from time to time since September 2015.

  • TRB4: I partnered with a friend in a startup in the translation industry in San Francisco. We initially hired an external company to code our webservice (a search engine for existing translations), then I took over the programming duties and rebuilt and finished the site (now defunct) using PHP and Oracle.
